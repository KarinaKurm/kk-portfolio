MobilPoint Software

MobilPoint is a marketing software product which I helped build as a part of RadiantGroup team. As a lead designer and the only designer at this start-up, I was responsible for all UX/UI work as well as company website and all the graphic design and animation.

MobilPoint is a full solution which consists of the mobile-app builder, analytics and physical touch points.

MobilPoint technology is currently being used by large companies such as Johnson&Johnson, TGI Fridays, Indeed Labs and more.





  • Client: Radiant Group Media
  • Date: 2015
  • Service: Product Design

DrinkUp App

DrinkUp is an app that allows you to send an alcoholic beverage of your choice to a friend. A friend gets a certificate which he can redeem at the participating bars in return for a drink you gifted him.

I created UI for this app. The challenge was to create a style which would be casuall and classy at the same time.

  • Client: DrinkUp Canada
  • Date: 2015
  • Service: UI Design

Viral Game

Viral is a cross platform game where players play as Lupov, a deadly alien virus, whose main objective is to infect different living organisms in its quest to find the perfect host. Players must manoeuver through DNA gates within each organelle as they look out for dangerous anti-bodies and proteins. The varying environments inside these organs also affect the virus and must be taken into consideration when going through each level.

UNITY3D Development- Karina Kurmanbayeva
Animation and Illustation - Afonso Negramonte
Visual Design - Mudassir Iqbal
